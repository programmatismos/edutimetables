# EduTimetables Design Direction

## Brand
- Name: EduTimetables
- Tagline: Πρόγραμμα Ενδοσχολικών Εξετάσεων

## Colors
- Primary: #1E3A8A (deep blue)
- Accent: #3B82F6 (blue)
- Accent light: #DBEAFE
- Success: #10B981
- Warning: #F59E0B
- Danger: #EF4444
- Background: #F8FAFF (off-white with blue tint)
- Surface: #FFFFFF
- Border: #E2E8F0
- Text primary: #1E293B
- Text secondary: #64748B
- Text muted: #94A3B8

## Typography
- Font family: Poppins (Google Fonts)
- Display: 600-700 weight
- Body: 400-500 weight
- Small/label: 400 weight
- Generous line-height: 1.6

## Layout
- Left sidebar navigation (fixed, 240px)
- Main content area with padding
- Top header bar with context info
- Card-based content sections
- Clean table grids with hover states

## Spacing
- Base unit: 4px
- Section padding: 24px
- Card padding: 20px

## Style
- Clean, minimal, professional
- Sharp edges (rounded-md max)
- Subtle shadows on cards
- Color-coded exam slots (by subject type)
- Status badges with colored dots

## Components
- Sidebar with icon + label nav items
- Data tables with sorting/filtering
- Modal dialogs for CRUD
- Drag & drop exam grid
- Badge chips for roles/types
- Progress indicators for scheduling
