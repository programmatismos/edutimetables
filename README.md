# EduTimetables — Σύστημα Εξεταστικού Προγράμματος

Εφαρμογή δημιουργίας και διαχείρισης εξεταστικών προγραμμάτων για σχολικές μονάδες δευτεροβάθμιας εκπαίδευσης. Ο αλγόριθμος τοποθετεί αυτόματα εξετάσεις σε χρονοθυρίδες σεβόμενος υποχρεωτικούς κανόνες (αργίες, διαθεσιμότητα εκπαιδευτικών, συγκρούσεις τάξεων) και βελτιστοποιώντας μαλακές προτιμήσεις (σειρά τάξεων, ανάσες, ισοκατανομή επιτήρησης). Μετά τη δημιουργία, το πρόγραμμα εξάγεται σε PDF, Word, Excel ή CSV.

## 1. Απαιτήσεις & Εκκίνηση

**Απαιτήσεις:**
- [Node.js](https://nodejs.org/) ≥ 18 ή [Bun](https://bun.sh/) ≥ 1.0
- Σύγχρονος browser (Chrome, Firefox, Edge)

**Εκκίνηση (development):**

```bash
git clone <repo-url> edutimetables
cd edutimetables
bun install
bun run dev
```

Ανοίξτε `http://localhost:5173` στον browser σας.

**Build για παραγωγή:**

```bash
bun run build
bun run start
```

## 2. Βήματα Πρώτης Χρήσης

Ακολουθήστε τα παρακάτω βήματα **με αυτή τη σειρά**. Κάθε βήμα εξαρτάται από το προηγούμενο.

1. **Ρυθμίστε το Σχολείο** — Ορίστε όνομα, τύπο σχολείου, εξεταστική περίοδο και βάρδιες (πρωινή/απογευματινή ή και τις δύο).
2. **Προσθέστε Εκπαιδευτικούς** — Εισάγετε τους εκπαιδευτικούς με κωδικό ειδικότητας (π.χ. ΠΕ02) και σημειώστε αναστολές/αδυναμίες.
3. **Ορίστε Τάξεις & Τμήματα** — Δηλώστε τάξεις, τμήματα και αριθμό μαθητών ανά τμήμα. Τμήματα με >25 μαθητές μπορούν να χωριστούν.
4. **Εισάγετε Μαθήματα** — Συνδέστε κάθε μάθημα με εκπαιδευτικό (εισηγητή) και ορίστε τυχόν επιτηρητές.
5. **Δημιουργήστε Πρόγραμμα** — Ο αλγόριθμος τοποθετεί αυτόματα τις εξετάσεις. Μπορείτε να κάνετε χειροκίνητες αλλαγές με drag & drop.
6. **Εξάγετε** — Κατεβάστε το πρόγραμμα σε PDF, Word, Excel ή CSV.

## 3. Οδηγός Σελίδων

### Σχολείο (`/school`)

| Πεδίο | Περιγραφή |
|---|---|
| Όνομα σχολείου | Εμφανίζεται στις εξαγωγές |
| Τύπος σχολείου | ΓΕΛ, ΕΠΑΛ, ΓΕΛ/ΕΠΑΛ, ΓΥΜΝΑΣΙΟ, ΣΑΕΚ |
| Περίοδος εξετάσεων | Ημερομηνία έναρξης και λήξης |

| Αργίες / Αναστολές | Ημέρες χωρίς εξετάσεις (εθνικές αργίες, λοιπές) |

> **Tip:** Βεβαιωθείτε ότι η περίοδος εξετάσεων καλύπτει αρκετές εργάσιμες ημέρες ώστε ο αλγόριθμος να έχει χώρο να τοποθετήσει όλες τις εξετάσεις.

### Τάξεις & Τμήματα (`/classes`)

- Ορίστε τάξεις (π.χ. Α', Β', Γ') και τμήματα (π.χ. Α1, Α2).
- Δηλώστε **αριθμό μαθητών** ανά τμήμα (χρησιμοποιείται για split).
- Για ΣΑΕΚ: οι τάξεις είναι Α1, Α2, Β1, Β2 (εξαμηνιαία δομή).

---

### Μαθήματα (`/subjects`)

- Κάθε μάθημα συνδέεται με:
  - **Τάξη / τμήμα**
  - **Εισηγητή** (ο εκπαιδευτικός που εξετάζει)
  - **Επιτηρητές** (ένας ή περισσότεροι)
- Μαθήματα Γενικής Παιδείας και Ειδικότητας χειρίζονται αυτόματα βάσει της ειδικότητας του εισηγητή.

---

### Πρόγραμμα (`/schedule`)

- Πατήστε **«Αυτόματη Δημιουργία»** για να τρέξετε τον αλγόριθμο.
- Το αποτέλεσμα εμφανίζεται σε **πλέγμα ημέρα × βάρδια**.
- Κάθε κελί δείχνει: μάθημα, τάξη/τμήμα, εισηγητή, επιτηρητές.
- Μπορείτε να **μετακινήσετε** εξετάσεις χειροκίνητα.
- Συγκρούσεις σημαίνονται με **κόκκινο πλαίσιο** και badge `⚠ N`.

## 4. Κανόνες Αλγορίθμου

Ο αλγόριθμος εφαρμόζει δύο κατηγορίες κανόνων:

### Σκληροί Κανόνες — Δεν παραβιάζονται ποτέ

| Κωδικός | Κανόνας | Αιτιολογία |
|---|---|---|
| **H1** | **Αργίες / Αναστολές** | Σε ημέρα που έχει οριστεί ως αργία ή αναστολή, δεν τοποθετείται καμία εξέταση. |
| **H2** | **Αδυναμία Εκπαιδευτικού** | Εκπαιδευτικός που έχει δηλωμένη αδυναμία για συγκεκριμένη ημέρα δεν χρησιμοποιείται (ούτε ως εισηγητής ούτε ως επιτηρητής). |
| **H3** | **Μία Εξέταση ανά Τάξη/Βάρδια** | Στην ίδια βάρδια (π.χ. πρωινή Δευτέρας) δεν μπορούν δύο εξετάσεις του ίδιου τμήματος να συμπέσουν. |
| **H4** | **Μία Εξέταση ανά Τάξη/Ημέρα** | Το ίδιο τμήμα δεν εξετάζεται σε δύο διαφορετικές βάρδιες της ίδιας ημέρας. |
| **H5** | **Εισηγητής σε Διπλή Εξέταση** | Ο ίδιος εισηγητής δεν μπορεί να παρουσιάσει/εξετάζει δύο διαφορετικά τμήματα στην ίδια βάρδια. |
| **H6** | **Εισηγητής ≠ Επιτηρητής ταυτόχρονα** | Εκπαιδευτικός που είναι εισηγητής σε εξέταση δεν μπορεί ταυτόχρονα να ορίζεται επιτηρητής σε άλλη εξέταση. |

> **Σημείωση:** Εάν ο αλγόριθμος δεν μπορεί να τοποθετήσει εξέταση χωρίς να παραβιάσει σκληρό κανόνα, αναφέρεται σύγκρουση στο πρόγραμμα.

---

### Μαλακοί Κανόνες — Βελτιστοποιούνται, δεν είναι υποχρεωτικοί

| Κωδικός | Κανόνας | Περιγραφή |
|---|---|---|
| **S1** | **Γ' Τάξη Πρώτα** | Προτεραιότητα στην τελευταία τάξη (Γ' ΓΕΛ, Γ' ΕΠΑΛ) — εξετάζεται στις πρώτες ημέρες της εξεταστικής. |
| **S2** | **Ανάσα μεταξύ Εξετάσεων** | Αποφεύγονται συνεχόμενες ημέρες εξέτασης για το ίδιο τμήμα (τουλάχιστον μία ελεύθερη μέρα ανάμεσα). |
| **S3** | **Συγκέντρωση Ειδικοτήτων** | Μαθήματα ειδικότητας της ίδιας τάξης προτιμάται να εξετάζονται την ίδια ημέρα (διαφορετικές βάρδιες). |
| **S4** | **Ισοκατανομή Επιτήρησης** | Η επιτήρηση κατανέμεται ισόρροπα μεταξύ εκπαιδευτικών. Εξέταση 3 ωρών μετράει ως 1,5× σε σχέση με 2ωρη. |
| **S5** | **Αποφυγή Διπλής Επιτήρησης** | Ίδιος εκπαιδευτικός δεν επιτηρεί σε πρωινή ΚΑΙ απογευματινή βάρδια της ίδιας ημέρας, αν είναι δυνατόν. |

## 5. Σύστημα Προειδοποιήσεων (Warnings)

Το σύστημα ανιχνεύει συγκρούσεις **σε πραγματικό χρόνο** και τις εμφανίζει απευθείας στο πρόγραμμα.

### Πώς εμφανίζονται

- **Κόκκινο πλαίσιο** γύρω από το κελί της εξέτασης με σύγκρουση
- **Badge `⚠ N`** (N = αριθμός συγκρούσεων) στην επάνω δεξιά γωνία του κελιού
- **Inline μηνύματα** κάτω από το κελί με ακριβή περιγραφή του προβλήματος

### Τύποι Συγκρούσεων

| Κωδικός | Μήνυμα | Τι σημαίνει |
|---|---|---|
| H1 | «Η ημέρα έχει οριστεί ως αργία» | Εξέταση τοποθετήθηκε σε αργία — μετακινήστε τη |
| H2 | «Ο εισηγητής έχει αδυναμία» | Εισηγητής δεν είναι διαθέσιμος εκείνη την ημέρα |
| H3 | «Το τμήμα έχει ήδη εξέταση αυτή τη βάρδια» | Δύο εξετάσεις ίδιου τμήματος την ίδια βάρδια |
| H4 | «Το τμήμα έχει ήδη εξέταση αυτή την ημέρα» | Δύο εξετάσεις ίδιου τμήματος την ίδια ημέρα |
| H5 | «Ο εισηγητής εξετάζει αλλού αυτή τη βάρδια» | Εισηγητής σε δύο εξετάσεις παράλληλα |
| H6 | «Ο εισηγητής είναι επιτηρητής αλλού αυτή τη βάρδια» | Διπλός ρόλος εισηγητή/επιτηρητή ταυτόχρονα |
| — | «Ο επιτηρητής X εποπτεύει αλλού αυτή τη βάρδια» | Επιτηρητής ορίστηκε σε δύο εξετάσεις παράλληλα |

> **Τι να κάνω αν δω warning;** Μετακινήστε την εξέταση σε διαφορετική βάρδια/ημέρα, ή αλλάξτε τον εισηγητή/επιτηρητή από τη σελίδα Μαθήματα.

## 6. Τύποι Σχολείων

| Τύπος | Περιγραφή | Τάξεις |
|---|---|---|
| **ΓΕΛ** | Γενικό Λύκειο | Α', Β', Γ' |

## 7. Εξαγωγές

Από τη σελίδα **Εξαγωγές** (`/exports`) διατίθενται τέσσερις μορφές:

| Μορφή | Περιγραφή | Χρήση |
|---|---|---|
| **PDF** | Εκτυπώσιμο πρόγραμμα, οριζόντιος A4 | Ανάρτηση στο σχολείο, αποστολή |
| **Word (.docx)** | Επεξεργάσιμο έγγραφο, πίνακας A4 οριζόντιος | Τροποποίηση πριν την ανάρτηση |
| **Excel (.xlsx)** | Δύο φύλλα: «Πρόγραμμα» + «ανά Εκπαιδευτικό» | Διοικητική χρήση, αρχειοθέτηση |
| **CSV** | Κωδικοποιημένα δεδομένα, UTF-8 | Εισαγωγή σε άλλα συστήματα |

### Φύλλα Excel

- **Πρόγραμμα:** Όλες οι εξετάσεις ανά ημέρα/βάρδια (μάθημα, τμήμα, εισηγητής, επιτηρητές)
- **ανά Εκπαιδευτικό:** Ο φόρτος επιτήρησης ανά εκπαιδευτικό (συνολικές ώρες, ημέρες)

### Εκτύπωση PDF

Χρησιμοποιήστε το κουμπί **«Εκτύπωση / PDF»** — ο browser ανοίγει το διάλογο εκτύπωσης σε **οριζόντιο A4**. Επιλέξτε «Αποθήκευση ως PDF» για αρχείο.

## 8. Split Τμημάτων

Τμήματα με μεγάλο αριθμό μαθητών μπορούν να χωριστούν σε υποτμήματα για την εξέταση:

| Συνθήκη | Αποτέλεσμα |
|---|---|
| Αριθμός μαθητών > 25 | Προτείνεται αυτόματα split |
| Επιλογή «Force Split» | Split ανεξαρτήτως αριθμού μαθητών |

Τα υποτμήματα (π.χ. Β1α, Β1β) εξετάζονται σε **διαφορετικές βάρδιες** της ίδιας ημέρας ή σε διαφορετικές ημέρες, ανάλογα με τη διαθεσιμότητα.

> Κανόνας H3/H4 ισχύει και για τα υποτμήματα ξεχωριστά — δεν αντιμετωπίζονται ως ένα τμήμα μετά το split.

## Συχνές Ερωτήσεις

**Ο αλγόριθμος δεν τοποθέτησε όλες τις εξετάσεις. Τι κάνω;**
Ελέγξτε: (α) αν η περίοδος εξετάσεων έχει αρκετές ημέρες, (β) αν υπάρχουν πολλές αδυναμίες εκπαιδευτικών, (γ) αν τα τμήματα έχουν υπερβολικά πολλά μαθήματα για το διαθέσιμο χρονικό πλαίσιο.

**Μπορώ να αλλάξω χειροκίνητα το πρόγραμμα;**
Ναι. Μετά τη δημιουργία, μπορείτε να μετακινείτε εξετάσεις. Τα warnings ενημερώνονται αμέσως.

**Πώς εξάγω μόνο το πρόγραμμα ενός τμήματος;**
Χρησιμοποιήστε το CSV και φιλτράρετε στο Excel, ή χρησιμοποιήστε το φύλλο «ανά Εκπαιδευτικό» στο αρχείο Excel.

**Τι γίνεται αν αλλάξω αργία μετά τη δημιουργία;**
Το πρόγραμμα δεν ξαναδημιουργείται αυτόματα. Θα εμφανιστεί warning H1 στις επηρεαζόμενες εξετάσεις — μετακινήστε τες χειροκίνητα ή τρέξτε ξανά τον αλγόριθμο.
